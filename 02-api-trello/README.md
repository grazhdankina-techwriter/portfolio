# 📊 Trello API: Как получить список задач (карточек) с доски

> 📋 **Паспорт проекта**
> * **Цель:** Показать умение работать с технической документацией, API, JSON и блоками кода.
> * **ЦА:** Начинающие разработчики или продвинутые пользователи Trello.
> * **Инструменты:** Markdown, cURL, Postman.

Этот туториал покажет, как с помощью REST API Trello получить JSON-список всех карточек с конкретной доски.

## 📌 Prerequisites (Что нужно подготовить)

Прежде чем начать, убедитесь, что у вас есть:
1. Аккаунт в Trello.
2. Созданная доска с несколькими карточками.
3. Установленный [Postman](https://www.postman.com/) или умение работать с `cURL` в терминале.

---

## Шаг 1. Получение API Key и Token

Для запросов нам нужны учетные данные.
1. Перейдите на страницу [Trello Power-Ups](https://trello.com/power-ups/admin) и создайте новый Power-Up (или используйте существующий).
2. Нажмите **Generate a new API Key**. Скопируйте **API Key**.
3. Рядом с ключом нажмите на ссылку для генерации **Server Token**. Разрешите доступ и скопируйте токен.

> 🔒 **Безопасность:** Никогда не коммитьте API-ключи и токены в публичные репозитории (GitHub). Используйте переменные окружения.

---

## Шаг 2. Получение ID доски

Чтобы запросить карточки, нужно знать ID доски.
1. Откройте нужную доску в браузере.
2. В конце URL вы увидите ID (например: `trello.com/b/aBcD1234/moya-doska`).
3. Скопируйте этот ID (`aBcD1234`).

---

## Шаг 3. Выполнение GET-запроса

Отправьте запрос к эндпоинту `cards`.

### Вариант А: Через cURL (Терминал)
```bash
curl --request GET \
  --url 'https://api.trello.com/1/boards/{BOARD_ID}/cards?key={API_KEY}&token={TOKEN}'
Замените {BOARD_ID}, {API_KEY} и {TOKEN} на ваши значения.
Вариант Б: Через Postman
Создайте новый GET запрос.
Вставьте URL: https://api.trello.com/1/boards/{BOARD_ID}/cards
Во вкладке Params добавьте два ключа:
Key: key | Value: ваш_API_ключ
Key: token | Value: ваш_токен
Нажмите Send.
📥 Ожидаемый результат (Response)
При успешном запросе (статус 200 OK) вы получите массив JSON. Пример одной карточки:
